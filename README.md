# BNSF

BNSF Railway, a subsidiary of Berkshire Hathaway Inc., is one of the largest freight railroad networks in North America. The company operates an extensive network of over 32,000 route miles in 28 states and three Canadian provinces. BNSF provides a comprehensive Customer API suite enabling shippers and logistics partners to integrate freight operations programmatically.

**URL:** [View APIs.yml](https://raw.githubusercontent.com/api-evangelist/bnsf/refs/heads/main/apis.yml)

## APIs

### BNSF Tracing API
Real-time shipment tracking from origin to destination for automotive VINs, carload railcars, intermodal units, and trains. Supports bulk queries of up to 300 vehicles or units per request.

### BNSF Hub Operations API
Access to intermodal facility data including container/trailer delivery details, storage locations, driver pickup/delivery, dray bookings, and gate operations.

### BNSF Pricing & Rates API
Programmatic access to freight shipping prices and rates for both carload and intermodal shipments.

### BNSF Schedules API
Intermodal transit schedules for planning freight shipment timing across the BNSF rail network.

### BNSF Waybill Management API
Electronic submission and retrieval of bills of lading with transit details for carload shipments.

### BNSF Reference Files API
Reference data including city names, commodity descriptions (STCC codes), station data, event codes, and hazardous materials information.
